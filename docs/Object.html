(static) copy(src, deepopt) → {Object}

Creates a copy of the given object. By default the clone function performs a deep copy of the object, passing deep = false will only clone the properties directly owned by the object are transfer to the object includinig Date objects. NOTE: clone is replaceable, copy is not since is a static method.
