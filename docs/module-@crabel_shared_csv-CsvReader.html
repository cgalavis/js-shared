    new CsvReader(file_name)
    csv.js, line 19

    Simple class to load CSV file. The first row is used as the header, values of other rows can be access via the column name provided in the header.
    Parameters:
    Name Type Description
    file_name String Name of the file to load.

    doc :Array

    Array of rows from the loaded CSV document. Each cell can be access by row index and column name, i.e. sl.doc[0].name.
