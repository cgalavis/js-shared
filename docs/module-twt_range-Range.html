    new Range(p_from, {string)
    range.js, line 24

    The range object supports range style operations on any number of intervals. The constructor takes two parameters each represent the property names of the two ends of an interval object. Any object can be used as an interval as long as it supports these two properties.
    Parameters:
    Name Type Description
    p_from string Name of the property that holds the left side of an interval.
    {string p_to Name of the property that holds the right side of the interval.
