    Encapsulates list of non overlapping intervals. The interval object can be any object, the extraction functions must be specified, these function return a comparable value such as a string or a number. Specialized "Range' classes can be defined by providing extraction function for the specific object type. The class RangeArray is an example of how Range specialization.
