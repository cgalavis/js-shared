Range

@crabel/shared/range~ Range

new Range(p_from, p_to)

Source:
  • range.js, line 25
The range object supports range style operations on any number of intervals. The constructor takes two parameters each represent the property names of the two ends of an interval object. Any object can be used as an interval as long as it supports these two properties.
Parameters:
Name Type Description
p_from string Name of the property that holds the left side of an interval.
p_to string Name of the property that holds the right side of the interval.

validInterval(int) → {boolean}

Source:

Validates the given interval. If array it must have 2 elements and the forst must be less or equal to the seconds. If object it must support the two interval properties defined in the range and the left side (from) must be less or equal to the right side (to). This function is used by other methods to verify the validity of input parameters.
